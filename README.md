
# Castel Brain — MVP 2

## Instalação

1. Instale o Python 3.11 ou superior.
2. Abra o terminal dentro desta pasta.
3. Execute:

```bash
pip install -r requirements.txt
```

4. Depois rode:

```bash
streamlit run app.py
```

## Estrutura esperada da planilha

A planilha pode conter os seguintes campos:

- Data
- Empresa
- Produto
- Imobiliaria
- Gerente
- Corretor
- Origem
- Leads
- Visitas
- Propostas
- Vendas
- VGV

O sistema permite mapear manualmente os nomes das colunas.
